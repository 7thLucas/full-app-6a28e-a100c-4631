# WAKE UP — Mobile Alarm App

## Product
**WAKE UP** is a mobile alarm application built on a cheerful, cute 3D aesthetic. It features a character-driven UI anchored by an adorable yellow dinosaur mascot, transforming the daily alarm-setting routine into a joyful, personality-filled morning experience.

## Brand & Visual Identity
| Attribute | Value |
|---|---|
| **App Name** | WAKE UP |
| **App Name Typography** | Bold font, dark brown |
| **Primary Color** | `#FFD500` (vibrant yellow) |
| **Secondary Color** | `#FFF9E2` (soft cream) |
| **Background Gradient** | `#FFD500` → `#FFF9E2` |
| **Mascot** | Cute yellow 3D dinosaur with small claws — waving in the header logo; holding analog clock hands on the dashboard |
| **Tone** | Cheerful · Cute · Playful · Friendly · Warm |

## Dashboard UI Design

### Header
- Yellow dinosaur mascot logo (waving pose) displayed at the top of the screen
- "WAKE UP" app name in bold dark brown font below the mascot logo

### Hero Element — Analog Clock
- Large round analog clock, light orange color
- Cute sleepy facial expression (eyes closed, mouth open) on the clock face
- Numbers 1–12 clearly displayed around the face

### Mascot Interaction
- Yellow dinosaur character stands beside the analog clock
- Uses its small-clawed hands to hold both the hour and minute hands
- Visually conveys and gamifies the alarm-setting action

### Supporting UI Elements
- Large digital time display prominently shown (e.g. **07:30 AM**)
- Prominent yellow **"SET ALARM"** button positioned below the clock
- Background: blurred bedroom scene with pastel-colored bed and pillows

## Positioning
A fun-first mobile alarm app that replaces cold, utilitarian alarm experiences with a warm, character-driven design. Stands out through its 3D aesthetic, lovable dinosaur mascot, and intuitive one-tap alarm flow that makes users actually want to engage with it every morning.

## Target Audience
Mobile users — particularly younger demographics — who value personality, visual delight, and character-driven design in their everyday productivity apps.
